\doxysubsection{Detailed Description}
Interface (classe base abstrata) para os modelos de permeabilidade relativa. 

Esta classe define o "{}contrato"{} que todos os modelos de Kr (Tabelado, Corey, etc.) devem seguir. Ela é a base do Padrão de Projeto Strategy, permitindo que a Calculadora opere em um alto nível de abstração (Inversão de Dependência).
