\doxysubsection{Detailed Description}
Classe utilitária que implementa o Padrão de Projeto Facade. 

Esta classe esconde toda a complexidade de lidar com o \doxylink{class_gnuplot}{Gnuplot} (criar arquivos temporários, formatar scripts, chamar o system()) por trás de um único método estático.
