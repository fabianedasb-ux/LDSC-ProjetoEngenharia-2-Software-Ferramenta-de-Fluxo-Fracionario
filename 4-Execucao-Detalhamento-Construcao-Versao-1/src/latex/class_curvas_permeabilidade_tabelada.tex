\doxysubsection{Detailed Description}
Implementação concreta da interface \doxylink{class_i_curvas_permeabilidade}{ICurvas\+Permeabilidade} para dados tabulados. 

Esta classe lê uma tabela de Sw, Krw e Kro de um arquivo de entrada e usa interpolação linear para calcular valores intermediários.
