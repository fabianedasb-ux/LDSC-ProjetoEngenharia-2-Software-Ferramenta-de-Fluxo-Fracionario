\doxysubsection{Detailed Description}
Classe principal que implementa o "{}domínio de negócio"{} (a lógica de engenharia). 

Esta classe é responsável por implementar a equação de Buckley-\/\+Leverett. Ela depende de uma abstração (\doxylink{class_i_curvas_permeabilidade}{ICurvas\+Permeabilidade}) e não de uma implementação concreta, seguindo o Princípio da Inversão de Dependência.
