Detailed Description

Implementação concreta da interface ICurvasPermeabilidade para o modelo de Corey.

Esta classe lê os 6 parâmetros do modelo de Corey do arquivo de entrada e calcula Krw/Kro analiticamente usando as fórmulas de Corey.
